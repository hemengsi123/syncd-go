## windows下文件夹同步工具。
```
自己使用的网盘(dropbox,坚果云)选择性同步都只限于文件夹，不能很好的精度同步数据，linux下有现成的工具(lsyncd)，windows下就没有用得顺手的工具，所以在 syncthing-inotify 的基础上修改，开发了这个工具，没经过严格的测试，能够满足个人使用
```

go get github.com/hemengsi123/syncd-go
